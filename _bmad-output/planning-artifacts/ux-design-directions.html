Minstrel — Design Direction Exploration

Six visual approaches for Minstrel's interface. Same design foundation (colors, typography, tokens), different layout strategies. Evaluate based on: layout intuitiveness, visual weight, canvas prominence, and alignment with the "professional readiness + instant accessibility" goal.

Direction 1: Immersive Canvas

Silent Coach Mode — Maximum visualization, floating overlays

The visualization canvas takes 100% of the viewport. UI elements float as translucent overlays with backdrop blur. Closest to the "flow state" ideal — the musician sees their music filling the screen. Data panels appear on hover or pause.

+ Maximum immersion + Flow state optimized + Dramatic first impression ~ Floating panels may feel unstable ~ Data access requires interaction

Direction 2: Split Dashboard

Dashboard + Chat Mode — Canvas left, data right

Classic split-panel layout. Visualization canvas takes 60% left, structured data panel takes 40% right. Clean separation of concerns. Most familiar layout pattern — immediately understandable. Epidemic Sound-style card organization in the sidebar.

+ Instantly understandable layout + Data always visible + Clean information hierarchy ~ Less canvas space than immersive ~ May feel like a generic dashboard

Direction 3: Card Grid

Modular layout — Everything in organized cards

Every piece of information lives in its own card. Canvas is the largest card. Metrics, insights, and controls are separate cards arranged in a grid. Most Epidemic Sound-aligned of all directions. Clean, modular, easy to scan.

+ Most Epidemic Sound-like + Clear information boundaries + Scannable at a glance ~ Canvas shares space with cards ~ May feel more like analytics than studio

Direction 4: Command Center

Three-panel layout — Data flanking the canvas

Canvas in the center, flanked by data panels on both sides. Left panel shows session/performance data. Right panel shows AI insights and chat. Cockpit-style layout. Most information-dense option.

+ Maximum data visibility + Symmetric, balanced feel + Everything accessible at once ~ Reduced canvas size ~ May overwhelm minimal-tech users ~ Dense for first-time experience

Direction 5: Minimal Focus

Silent Coach Mode — Almost pure canvas, mini data strip

The most minimal approach. Canvas takes nearly all space. A thin data strip at the bottom shows only essential metrics as compact mini-cards. Closest to "90% instrument, 10% screen." Maximum focus on the music, minimum UI distraction.

+ Maximum simplicity + Lowest cognitive load + Best for flow state ~ Very limited data visibility ~ May feel too sparse for data-driven users

Direction 6: Tabbed Workspace

Replay Studio Mode — Timeline-driven with detail tabs

Content-tab navigation below the mode switcher organizes detail views. Canvas left, tabbed detail panel right. Tabs allow switching between Insights, Chat, Progress, and Drills without losing canvas view. Best for information-rich modes like Replay Studio.

+ Organized depth + Tabs reduce overwhelm + Good for Replay Studio ~ Extra navigation layer ~ Tabs can feel like separate sections
